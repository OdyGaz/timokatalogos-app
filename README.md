# Δημιουργός Τιμοκαταλόγου (Menu Creator)

Μια desktop εφαρμογή βασισμένη στο **Electron** για τη διαχείριση, ζωντανή προεπισκόπηση και εξαγωγή τιμοκαταλόγων σε μορφή PDF (A4). 
Η εφαρμογή είναι πλήρως παραμετροποιήσιμη για οποιοδήποτε κατάστημα.

---

## 🛠️ Τεχνολογίες

* **Electron**: Για τη μετατροπή της web εφαρμογής σε αυτόνομη desktop εφαρμογή.
* **HTML5 & CSS3**: Δίστηλη διάταξη (flexbox) με ειδικό stylesheet (`print.css`) βελτιστοποιημένο για εκτύπωση χωρίς περιθώρια.
* **Vanilla JavaScript (ES6 Modules)**: Διαχωρισμός κώδικα σε αυτόνομα modules (State, UI, Utils) χωρίς εξωτερικά frameworks.
* **LocalStorage**: Αυτόματη αποθήκευση και ανάκτηση των δεδομένων τοπικά, ώστε να μην χάνονται οι αλλαγές κατά το κλείσιμο της εφαρμογής.
* **Electron Builder**: Για τη δημιουργία φορητού (portable) εκτελέσιμου αρχείου `.exe` για Windows.

---

## 🚀 Χαρακτηριστικά

1. **Ζωντανή Προεπισκόπηση (Live Preview)**: Κάθε αλλαγή στο όνομα, τις τιμές ή τις περιγραφές των προϊόντων εμφανίζεται αμέσως στην προσομοίωση της σελίδας Α4.
2. **Διαχείριση Σελίδων & Στηλών**: 
   * Υποστήριξη 3 αυτόνομων σελίδων (π.χ. Ροφήματα, Ποτά/Μεζέδες, Γλυκά).
   * Δυνατότητα προσθήκης, διαγραφής και αναδιάταξης (μετακίνηση πάνω/κάτω) κατηγοριών και προϊόντων.
3. **Ανίχνευση Υπερχείλισης (Overflow Warning)**: Αυτόματη ειδοποίηση με οπτική σήμανση εάν το περιεχόμενο μιας στήλης ξεπεράσει το φυσικό ύψος της σελίδας Α4, αποτρέποντας την αποκοπή κειμένου κατά την εκτύπωση.
4. **Προσαρμογή Φόντου**: Δυνατότητα εισαγωγής προσαρμοσμένης εικόνας φόντου (A4 background image) απευθείας από την εφαρμογή.
5. **Δημιουργία Αντιγράφων Ασφαλείας (Backup)**: Εξαγωγή και εισαγωγή ολόκληρου του καταλόγου σε αρχείο μορφής `.json`.
6. **Εξαγωγή PDF**: Άμεση παραγωγή αρχείου PDF με τη χρήση του native διαλόγου αποθήκευσης των Windows και βελτιστοποιημένες ρυθμίσεις εκτύπωσης φόντου.

## 📁 Αρχιτεκτονική Project

Η δομή των αρχείων του project ακολουθεί την παρακάτω διάταξη:

```text
/timokatalogos-app
├── index.html                 # Το βασικό σημείο εισόδου της εφαρμογής
├── main-electron.js           # Electron Main Process (Διαχείριση παραθύρου & έκδοσης PDF)
├── package.json               # Ρυθμίσεις project, scripts και εξαρτήσεις
├── .gitignore                 # Αποκλεισμός αρχείων από το Git (π.χ. node_modules)
├── assets/                    
│   └── images/                # Φόντο, εικονίδια και γραφικά στοιχεία
├── css/                       
│   ├── main.css               # Βασικές ρυθμίσεις layout (split-screen)
│   ├── editor.css             # Στυλ για τη φόρμα εισαγωγής (αριστερό panel)
│   ├── preview.css            # Στυλ για την εμφάνιση της Α4 (δεξί panel)
│   └── print.css              # Ρυθμίσεις που ενεργοποιούνται μόνο κατά την εκτύπωση
└── src/                       
    ├── main.js                # Εκκίνηση εφαρμογής & Event Listeners
    ├── state.js               # Διαχείριση δεδομένων, LocalStorage & Backup
    ├── ui.js                  # Δυναμική σχεδίαση (render) του DOM
    └── utils.js               # Βοηθητικές συναρτήσεις (μορφοποίηση τιμών)
```

## 💻 Οδηγίες Εγκατάστασης & Εκτέλεσης

Για να τρέξετε την εφαρμογή τοπικά σε περιβάλλον ανάπτυξης (development mode):

### Προαπαιτούμενα
Θα πρέπει να έχετε εγκατεστημένο το [Node.js](https://nodejs.org/) στον υπολογιστή σας.

### 1. Εγκατάσταση Εξαρτήσεων
Ανοίξτε το τερματικό στον φάκελο του project και εκτελέστε:
```bash
npm install
```

### 2. Εκτέλεση της Εφαρμογής
Για να εκκινήσετε την εφαρμογή μέσω του Electron:
```bash
npm start
```

## 📦 Δημιουργία Εκτελέσιμου Αρχείου (Build)

Η εφαρμογή είναι ρυθμισμένη να παράγει μια αυτόνομη (portable) έκδοση για Windows που δεν απαιτεί εγκατάσταση.

Για να δημιουργήσετε το αρχείο `.exe`:
```bash
npm run dist
```
Μόλις ολοκληρωθεί η διαδικασία, το εκτελέσιμο αρχείο θα βρίσκεται μέσα στον φάκελο **`dist/`** με το όνομα `Timokatalogos.exe`.
